Atue como um Desenvolvedor Front-end Senior e UI Designer. Crie o código para um site pessoal (Landing Page) estilo "Portfólio de Trajetória" para a Bruna, uma estudante de Medicina Veterinária.

DIRETRIZES DE DESIGN:

Estilo: "Fofo e Moderno". Use bordas arredondadas (rounded-3xl), sombras suaves, fontes elegantes (uma Serif para títulos e Sans para corpo) e elementos visuais que fujam do layout padrão de colunas retas.

Paleta de Cores: Tons pastéis modernos (Verde menta suave, areia e um toque de marrom terra bem claro para remeter a cavalos e natureza).

Tecnologia: React + Tailwind CSS (lucide-react para ícones).

ESTRUTURA DE CONTEÚDO (Escrito em 1ª Pessoa):

Hero: Apresentação da Bruna como futura Médica Veterinária na Anhanguera Sorocaba. Um título impactante que misture amor e ciência.

A Transição: Uma seção criativa contando que ela iniciou em Farmácia, mas percebeu que seu coração batia mais forte pela saúde animal, decidindo recomeçar na Veterinária.

Grandes e Pequenas Paixões: Um destaque especial para o amor dela por Cavalos (sua maior paixão) e cachorrinhos.

Minhas Companheiras: Uma seção tipo "card" ou "galeria" dedicada às suas duas cachorras, Luna e Mavis.

Jornada Acadêmica: Um pequeno badge ou linha do tempo indicando que ela está no 2º semestre na Anhanguera de Sorocaba.

REQUISITOS TÉCNICOS:

Layout Responsivo.

Efeitos de hover suaves nos cards.

Use componentes modernos (ex: uma seção em formato de 'Bento Grid' para as fotos e paixões).

O código deve ser limpo e pronto para uso.
